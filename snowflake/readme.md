Twitter Snowflake Docker Container
=

Goals
-

-	1.) Create a scriptable dockerfile that initializes snowflake successfully on ubuntu:12.04 
-	2.) Allow dynamic creation and port changing
-	3.) Include a haproxy configuration script to help run this behind a proxy
